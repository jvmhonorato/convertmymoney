## ConvertMyMoney




Application to convert currency


## Technology 

Here are the technologies used in this project.

* NODE JS
* HTML
* CSS





## Services Used

* Github





## Getting started

* Dependency
  - Express JS
  - Axios
  - EJS
 
  
  
* To install the dependencies.
```bash
$  npm install
```
  
  
* To run the project.
```bash
$ npm  start
```
